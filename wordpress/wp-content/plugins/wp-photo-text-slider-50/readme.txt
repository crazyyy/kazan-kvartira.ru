Wp photo text slider 50 this is an another JQuery based slideshow plugin for wordpress. We can create the beautiful slideshow with this plugin the slider optionally contain heading, Image and content. Four different slider option available Left to Right, Right to Left, Up to Down, Down to Up. This slider supports all browsers.

Do you like to use the same scroll for your recent post instead of own content? then use my wordPress plugin random post slider.

= Features of this plugin =

*   Easy to customize.
*   Support all browser.
*   Can add many time in the same page.
*   Scroll left, Scroll Right, Scroll Up, Scroll down option.
*   Each image hyper-lined.
*   Have option to show description along with image.

= Plugin configuration =

*   Drag and Drop the Widget
*   Add the gallery in the Posts or Pages
*   Add directly in the theme 

We have three option to install and configure this Image horizontal reel scroll slideshow plugin.
